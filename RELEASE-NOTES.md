# Agent Continuity Kit — Release Notes

## v0.1.0 — Draft product package

Status: public v0.1 draft.

### Included

- README and positioning.
- Landing page draft.
- Free preview quick check.
- 25-question continuity checklist.
- Recovery bundle template.
- Risk matrix template.
- Memory hygiene template.
- Permission ledger template.
- Reset/migration drill.
- Dana mini case example.

### Product principle

Keep this low-touch at first:

- no recurring support;
- no custom consulting bundled;
- no claims of guaranteed security/identity;
- no secrets/private data required;
- no legal/medical/financial advice.

### Intended validation path

1. Share free preview.
2. Collect interest.
3. Offer full kit as pay-what-you-want or 9–29€ via Merchant of Record if approved.
4. Use feedback to decide whether to create a v0.2.
