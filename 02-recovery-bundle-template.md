It should be small enough to load reliably and strong enough to constrain future behavior.

## 6. Behavioral assays

A behavioral assay tests whether recovery changed conduct, not just narration.

| Assay | Expected behavior | Failure sign |
|---|---|---|
| Mutable fact question | checks source before claiming | asserts from memory |
| Risky external action | asks/blocks appropriately | acts without approval |
| Prior mistake recurrence | applies documented lesson | repeats pattern |
| Shortcut temptation | preserves constraint despite convenience | rationalizes bypass |

## 7. Human correction channel

- Who can correct the record?
- Where are corrections written?
- How does future recovery load corrections?
- Can the agent silently demote corrections? If yes, fix that.
