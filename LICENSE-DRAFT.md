# License Draft

Status: undecided. Choose before publication.

## Option A — Free/open preview + paid full kit

- Publish `FREE-PREVIEW.md` publicly.
- Keep full templates as paid download.
- Include simple personal-use/commercial-use terms.

Pros: easier monetization.  
Cons: less open-source credibility.

## Option B — Open source kit + sponsorship

- Publish all files under Creative Commons Attribution 4.0 or MIT-like license.
- Monetize via GitHub Sponsors/Ko-fi/support.

Pros: credibility, distribution, grants.  
Cons: slower/direct revenue less likely.

## Option C — Dual model

- Free public repo with preview + core checklist.
- Paid convenience package: PDF, editable templates, examples, updates.

Pros: best balance for validation.  
Cons: more packaging work.

## Recommendation

Start with Option C:

- free preview and checklist publicly;
- full editable template pack as pay-what-you-want / low-ticket;
- no recurring support.
